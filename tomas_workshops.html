Smart Business

Impact van technologie

Tijdens de aller eerste les van Smart Business hebben we direct kennis gemaakt met wat bepaalde digitaliseringen kunnen doen met de industrie en economie. In groepen van vier personen hebben we ieder een eigen video gekregen welke op een manier te maken heeft met slimme technologiën. Aan ons was per groep de opdracht om in een korte tijd te achterhalen op welke manier de vier video's aan elkaar gelinkt zijn.

Wij zijn als groep te werk gegaan door in een paar korte woorden de video te omschrijven, vervolgens zijn wij in gesprek gegaan over wat de connectie zou kunnen zijn. In de figuur hieronder hebben we deze steekwoorden genoteerd, vanuit die steekwoorden zijn we tot vier woorden gekomen die alle vier de video's samenvatten. Tot slot hebben we een korte mini presentatie gegeven over welke connectie bij ons te zien was. In het geval van onze video's is volgens ons de connectie het veranderen van het werk door de jaren heen. Onderandere door de komst van het corona virus word er steeds meer thuis gewerkt, dit is een blijvende verandering ook na de lockdown van afgelopen jaren. Er wordt dan ook steeds meer gestreefd naar het automatiseren van het werk om hierdoor thuiswerken mogelijk te maken en een grotere efficientie te realiseren.

HTML en CSS

In de eerste les van Smart Connection hebben we voor het eerst kennis gemaakt met het schrijven in de taal HTML en CSS. Hierbij zijn we begonnen met het maken van een site zonder het gebruik van CSS opmaak, hierna hebben we een CSS opmaak toegevoegd aan deze site. Hierdoor is het erg duidelijk om te zien wat de CSS doet. Ik had voorafgaand aan deze minor nog nooit gebruik gemaakt van HTML of CSS of iets wat hier op lijkt. Ik vond het persoonlijk erg indrukwekkend om te zien wat ik na één les al had bereikt. In basis taal kan ik een site vanuit niks maken, daarnaast kan ik deze vorm geven doormiddel van kleuren en afbeeldingen toe te voegen.

Application Programming Interface

Application Programming Interface ook wel API genoemd is bedoeld om op een geformaliseerde manier gegevens uit te wisselen tussen apps. Simpel gezegd zorgen API's er voor dat men gegevens(foto's, berichten, meldingen etc.) kan delen. Voor het bekijken van zulke gegevens worden echter ook API's voor gebruikt. Je kan hierbij denken aan sociale media apps, voor het uitwisselen van de gegevens die men stuurd worden API's gebruikt.

De term API is één van de nieuwe termen die ik gedurende deze minor heb geleerd en vind het een erg interessant term omdat deze uitlegt hoe het uitwisselen van data in het dagelijkse leven werkt. Naast de betekenis van de term heb ik hier ook mee leren werken, ik heb geleerd hoe men bepaalde data op kan vragen. Zo ben ik te werk gegaan met het simpele voorbeeld van het opvragen van het weer op op een bepaalde locatie. Hierna ben ik aan de gang gegaan met de foto van de dag van NASA.
